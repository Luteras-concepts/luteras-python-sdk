# Changelog

All notable changes to the LUTERAS Python SDK will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/)
and this project follows [Semantic Versioning](https://semver.org/).

---

## [1.0.0] - 2026-07-02

### Added

- Initial public release of the LUTERAS Python SDK.
- Official PyPI package (`pip install luteras`).
- License creation API.
- License verification API.
- API Key authentication.
- Structured exception handling.
- Billing module.
- Dashboard module.
- API Keys module.
- Onboarding module.
- SDK documentation.
- Public GitHub repository.
- MIT License.
- Developer issue templates.
- Production-ready README.
